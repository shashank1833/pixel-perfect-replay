# Music

To set your own song as the default track:

1. Drop your MP3 file here and name it exactly: `birthday-song.mp3`
   (full path: `/app/public/audio/birthday-song.mp3`)
2. It will be served at `/audio/birthday-song.mp3` and load automatically.

You can also upload a song at runtime using the upload button in the Music section
(this plays your file for the current session via an in-browser object URL).
